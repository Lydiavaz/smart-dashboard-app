# smart-dashboard-app

A simple JavaScript dashboard with login and real-time data.

## Features
- Login using localStorage
- Welcome message
- Weather data (Open-Meteo API)
- Crypto prices (CoinGecko API)
- News (Hacker News API)
- Todos (JSONPlaceholder API)

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repo
   git clone https://github.com/your-username/smart-dashboard-app.git

2. Open the folder

3. Run index.html in browser

## Note
This project uses only frontend (no backend).
